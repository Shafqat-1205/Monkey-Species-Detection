# 🐒 Monkey Species Detection

A simple and basic deep learning project to classify different species of monkeys using Convolutional Neural Networks (CNNs). 

## 📋 Features
* **Custom CNN Models:** Implementation of a basic CNN from scratch to understand fundamental feature extraction.
* **Transfer Learning:** Integration of state-of-the-art models including `ResNet18`, `VGG16`, and others for improved accuracy.
* **Data Augmentation:** Techniques applied to prevent overfitting and improve model generalization.
* **Evaluation Metrics:** Includes accuracy tracking, loss curves, and [add any others, e.g., confusion matrix].


## 📊 Dataset
This project uses the **10 Monkey Species Dataset from Kaggle**. 
* It covers 10 different classes/species of monkeys.
*(Note: You can download the dataset [here](https://www.kaggle.com/datasets/slothkong/10-monkey-species))*


## 💻 Requirements
To run this project, you will need Python 3.x and the following libraries:

* `PyTorch`
* `numpy`
* `matplotlib`
* `opencv-python`
